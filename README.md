Ornek-Web-Sitesi-2
Frontend Mentor Web Uygulaması.
Bu proje, Frontend Mentor üzerinden alınan bir tasarımın React kullanılarak tamamen sıfırdan kodlanmış halidir. Tasarım referans olarak alınmıştır fakat tüm kodlama süreci kişisel gelişim amacıyla baştan sona kendim tarafından gerçekleştirilmiştir.

İçerik

-React ile komponent tabanlı yapı 
-Redux yerine props tabanlı yönetim yaklaşımı 
-Dark mode özelliği 
-Listeleme fonksiyonelliği 
-Responsive tasarım

Kullanılan Teknolojiler

-React 
-JavaScript (ES6+) 
-CSS 
-Vite
